The goal of the project is to use data from Azure DevOps (task management) and ETS (time management) systems to build reports that can help determine tasks, topics and problem domains 
where team members struggle the most to focus knowledge sharing meetings on that areas and help team members gain required skills and knowledge to improve productivity.
